### 장점
- 기존 코드를 건들이지 않고 새로운 타입의 인스턴스를 다른 방법으로 만들도록 확작이 가능하다. 
- 결합을 느슨하게 가져갔기 때문에

### 단점
- 클래스가 많아진다.
